# Olist 360: From Data to Decisions 📊

**E-Commerce Pricing & Churn Intelligence**  
*An end-to-end Python, SQL, Machine Learning, and Tableau analysis of 115,723 delivered orders from the Brazilian E-Commerce Public Dataset by Olist.*

---

## 📁 Repository Structure

- **`Data/`**: Dedicated folder for relational CSV dataset files and analytical data pipelines.
- **`Analysis Notebook/`**: Dedicated folder for Jupyter Notebooks (`.ipynb`) containing exploratory data analysis (EDA), Random Forest & Logistic Regression churn prediction models, and Facebook Prophet time-series revenue forecasting.
